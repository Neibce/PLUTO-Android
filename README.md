# PLUTO-Android
- 고등학교 동아리 팀코동(TeamCodingDongcheon 2018)의 비주얼 노벨 게임 제작 프로젝트 [PLUTO](https://github.com/Neibce/PLUTO)의 Android 포팅 버전
- Android(Java), Ren'Py, Google Play Game Services(업적, 리더보드), Firebase(FCM, Crashlytics)
